# Refactor Workflow

This workflow automates code refactoring tasks.

## Steps

1. **Analyze Code Structure**
   - Review current code organization
   - Identify refactoring opportunities
   - Check for code duplication

2. **Plan Refactoring**
   - Create refactoring plan
   - Identify files to modify
   - Plan step-by-step changes

3. **Apply Refactoring**
   - Execute refactoring changes
   - Maintain functionality
   - Update related files

4. **Run Tests**
   - Execute: `npm test`
   - Verify refactoring didn't break functionality
   - Ensure all tests pass

5. **Update Documentation**
   - Update code comments
   - Update README if needed
   - Document refactoring changes

## Usage

Invoke this workflow in Cascade using: `/refactor`
