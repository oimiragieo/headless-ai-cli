# Generate Tests Workflow

This workflow automates unit test generation.

## Steps

1. **Analyze Code Structure**
   - Review source files
   - Identify functions and classes to test
   - Understand code dependencies

2. **Generate Test Files**
   - Create test files for each source file
   - Generate test cases for all functions
   - Include edge cases and error handling

3. **Run Generated Tests**
   - Execute: `npm test`
   - Verify all tests pass
   - Fix any issues with generated tests

4. **Update Test Coverage**
   - Check test coverage
   - Generate coverage report
   - Ensure adequate coverage

## Usage

Invoke this workflow in Cline using: `/generate-tests.md`
